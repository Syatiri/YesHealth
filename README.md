## Run Locally

Clone the project

```bash
  git clone https://github.com/adwerygaming/YesHealth/
```

Go to the project directory

```bash
  cd YesHealth
```

Install dependencies

```bash
  flutter pub get
```

Build Your Own

```bash
  flutter build <apk/windows/web>
```
